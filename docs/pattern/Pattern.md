设计模式
======
看过设计模式很多次，但是每次看完，觉得自己懂了，但是实践中自己还是不会用。

因此有了这个项目，将设计模式重新捋一遍，分析使用场景，优缺点。会考虑将23种设计模式，编成一句顺口溜。

先做到手中有剑，心中有剑，才能达成手中无剑，心中无剑，人剑合一。
